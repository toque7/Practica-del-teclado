Práctica de Mecanografía

Ejemplo 1: Escribe el siguiente texto:

"La casa es bonita."

Ejemplo 2: Escribe el siguiente texto con signos de puntuación:

"¿Qué hora es?"

Ejemplo 3: Escribe la siguiente secuencia de números y signos:

"12345, 67890!"

Ejemplo 4: Escribe el siguiente texto con paréntesis:

"(El perro) juega."

Ejemplo 5: Escribe el siguiente texto con puntos y dos puntos:

"La hora es: 3:15 PM."

Ejemplo 6: Usa la tecla de tabulador para crear espacios:

"Nombre:\tJuan"

Ejemplo 7: Escribe el siguiente texto utilizando comillas:

"Dijo: 'Hola a todos.'"

Ejemplo 8: Escribe el siguiente texto utilizando el símbolo de porcentaje:

"El descuento es del 20%."

Ejemplo 9: Escribe el siguiente texto utilizando una dirección de correo electrónico:

"Contacto: correo@example.com"

Ejemplo 10: Escribe el siguiente texto con un símbolo de suma:

"2 + 2 = 4"

Ejemplo 11: Escribe el siguiente texto utilizando comillas dobles:

"El profesor dijo: \"Estudien mucho.\""

Ejemplo 12: Escribe el siguiente texto utilizando una barra invertida:

"C:\\Usuarios\\Juan"

Ejemplo 13: Escribe el siguiente texto utilizando una combinación de caracteres especiales:

"¡Hola, mundo! #Programación"

Ejemplo 14: Escribe el siguiente texto con una dirección web:

"Visita nuestro sitio en https://www.example.com"

Ejemplo 15: Escribe el siguiente texto utilizando puntos y comas:

"El perro, el gato, y el ratón; todos son amigos."
